Para qué sirve este proyecto

Para poder tener una biblioteca de componentes reutilizables y fáciles de copiar para sitios web.

La idea es tener inicialmente componentes sencillos hechos en HTML y CSS puro, con la opción de poder copiar su estructura, pero a futuro tener ese mismo componente en diferentes formatos, ya sea utilizando bootstrap o TailwindCSS como framework de CSS, formato de JS como función para reemplazar valores y renderizar muchos componentes, o hasta en frameworks como Angular o ReactJS

Además otra función que tendrá será su customización (leve, nada especial) en la misma página, pero que al cambiar dichos parámetros cambie para todos los componentes y pueda mantener la misma esencia.

En resumen, será una autoayuda para hacer sitios web super fáciles y rápidos. ¿En dónde optimiza el tiempo a diferencia de otros? En la función de copiar y pegar.

Y servirá de práctica, pues la meta es hacer un componente por día.
